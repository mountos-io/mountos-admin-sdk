# mountOS Admin SDK

Vendor-facing SDK for the mountOS Admin API. Wraps the appserv vendor API (`/api/v1/*`) with ED25519 JWT authentication.

## SDKs

| Language | Path | Package |
|----------|------|---------|
| [TypeScript](ts/README.md) | `ts/` | `@mountos/admin-sdk` (npm) |
| [Go](go/README.md) | `go/` | `github.com/mountos-app/mountos-admin-sdk/go` |
