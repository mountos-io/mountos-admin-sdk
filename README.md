# mountOS Admin SDK

Provider-facing SDK for the mountOS Admin API. Wraps the appserv provider API (`/api/v1/*`) with ED25519 JWT authentication.

## SDKs

| Language                   | Path  | Package                                              |
| -------------------------- | ----- | ---------------------------------------------------- |
| [TypeScript](ts/README.md) | `ts/` | `@mountos-app/admin-sdk` (GitHub / private registry) |
| [Go](go/README.md)         | `go/` | `github.com/mountos-app/mountos-admin-sdk/go`        |
